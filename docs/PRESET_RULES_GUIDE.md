# A股监控规则预设指南

## 概述

本系统根据A股市场的特点，精心设计了14个典型监控规则预设，涵盖涨跌停、价格波动、成交量异动、关键突破四大类场景。

## 规则分类

### 1. 涨跌停监控（4个规则）

#### 1.1 涨停预警（主板10%）
- **适用**：主板、中小板股票
- **触发条件**：涨幅接近10%
- **意义**：
  - 强烈买盘需求信号
  - 可能有重大利好消息
  - 连续涨停可能是热点题材或"妖股"
- **使用场景**：捕捉强势股，把握短期交易机会

#### 1.2 涨停预警（创业板/科创板20%）
- **适用**：创业板、科创板、北交所
- **触发条件**：涨幅接近20%
- **意义**：市场情绪极度亢奋
- **使用场景**：监控高成长性板块的极端行情

#### 1.3 跌停预警（主板10%）
- **适用**：主板、中小板股票
- **触发条件**：跌幅接近10%
- **意义**：
  - 重大利空消息
  - 可能存在财务问题
  - 市场恐慌性抛售
- **使用场景**：风险预警，及时止损

#### 1.4 跌停预警（创业板/科创板20%）
- **适用**：创业板、科创板、北交所
- **触发条件**：跌幅接近20%
- **意义**：严重风险信号，市场极度悲观
- **使用场景**：评估是否需要止损或抄底

---

### 2. 价格波动监控（4个规则）

#### 2.1 强势上涨（单日+5%以上）
- **触发条件**：单日涨幅超过5%
- **意义**：
  - 显著上涨信号
  - 通常伴随利好消息
  - 资金关注度提升
- **使用场景**：捕捉短期交易机会，关注是否有持续性

#### 2.2 接近涨停（单日+7%以上）
- **触发条件**：单日涨幅超过7%
- **意义**：
  - 极强势信号
  - 接近涨停板
  - 次日可能继续强势或开板分化
- **使用场景**：判断是否追涨或等待回调

#### 2.3 大幅下跌（单日-5%以上）
- **触发条件**：单日跌幅超过5%
- **意义**：
  - 可能有利空消息
  - 主力可能在出货
  - 技术面可能破位
- **使用场景**：风险警示，评估是否需要减仓

#### 2.4 剧烈震荡（单日波动8%以上）
- **触发条件**：盘中震荡幅度超过8%
- **意义**：
  - 多空激烈博弈
  - 分歧严重
  - 高风险高收益
- **使用场景**：适合短线高手，普通投资者应谨慎

---

### 3. 成交量异动（3个规则）

#### 3.1 放量突破（2倍量能）
- **触发条件**：成交量是过去1小时均量的2倍
- **意义**：
  - 量价配合
  - 可能突破关键位置
  - 资金进场信号
- **使用场景**：确认突破有效性，是买入信号

#### 3.2 巨量异动（5倍量能）
- **触发条件**：成交量是过去1小时均量的5倍
- **意义**：
  - 主力异动
  - 可能是建仓、拉升或出货
  - 需要结合价格分析
- **使用场景**：
  - 价格上涨+巨量 = 主力拉升
  - 价格下跌+巨量 = 主力出货
  - 价格横盘+巨量 = 主力建仓或换手

#### 3.3 持续放量（3倍量能30分钟）
- **触发条件**：成交量持续30分钟保持3倍以上
- **意义**：
  - 资金持续流入
  - 趋势延续性强
  - 不是瞬间冲高
- **使用场景**：确认趋势，增强持仓信心

---

### 4. 关键价位突破（2个规则）

#### 4.1 突破20元整数关口
- **触发条件**：价格突破20元
- **意义**：
  - 心理关口
  - 整数价位有吸引力
  - 突破后有惯性
- **使用场景**：低价股突破重要心理位，可能继续上涨

#### 4.2 突破50元整数关口
- **触发条件**：价格突破50元
- **意义**：
  - 中价股重要关口
  - 突破后打开上涨空间
  - 市场信心增强
- **使用场景**：中价股突破，可能进入加速上涨阶段

## 使用建议

### 组合使用策略

1. **保守型投资者**：
   - 涨停预警（捕捉强势股）
   - 大幅下跌（风险控制）
   - 跌停预警（止损保护）

2. **稳健型投资者**：
   - 强势上涨（关注优质标的）
   - 放量突破（确认买入时机）
   - 大幅下跌（风险警示）

3. **激进型投资者**：
   - 接近涨停（追涨机会）
   - 巨量异动（捕捉主力动向）
   - 剧烈震荡（短线交易）

4. **价值投资者**：
   - 跌停预警（寻找抄底机会）
   - 大幅下跌（评估买入时机）
   - 持续放量（确认趋势反转）

### 注意事项

1. **不要单独依赖一个指标**：应该结合多个规则综合判断
2. **考虑市场环境**：牛市和熊市中同样的信号意义不同
3. **关注基本面**：技术信号应该与基本面分析结合
4. **设置止损**：任何交易都应该有明确的止损位
5. **控制仓位**：不要因为一个信号就重仓单一股票

### A股特殊性

1. **T+1交易制度**：当日买入次日才能卖出，需要更谨慎
2. **涨跌幅限制**：主板10%，创业板/科创板20%
3. **停牌制度**：重大事项可能导致停牌
4. **ST制度**：连续亏损的股票有特殊标记
5. **龙虎榜**：大额交易会公开披露，可配合分析

- 可自定义修改参数
